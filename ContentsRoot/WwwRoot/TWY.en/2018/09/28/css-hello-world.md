# CSS "Hello, World!"

```HTML
<!DOCTYPE html>
<html lang="en-US">
    <head>
        <meta charset="utf-8">
        <title></title>
        <style>
            .hello::before {
                content: "Hello,";
            }
            .world::after {
                content: "World!";
            }
        </style>
    </head>
    <body>
        <span class="hello world">&nbsp;</span>
    </body>
</html>
```
