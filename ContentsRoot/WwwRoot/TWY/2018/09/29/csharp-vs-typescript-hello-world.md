# C# vs. TypeScript: "Hello, World!"


## C# "Hello, World!"

```CSharp
class HelloWorld
{
    static void Main(string[] args)
    {
        System.Console.WriteLine("Hello World!");
    }
}
```


## TypeScript "Hello, World!"

```TypeScript
console.log("Hello, World!");
```
